# 📅 Jadwal Pintarku Landing Page

Landing Page resmi untuk **Jadwal Pintarku**, aplikasi manajemen jadwal modern yang membantu pengguna mengatur aktivitas harian, jadwal pelajaran, tugas, dan pengingat secara lebih mudah, cepat, dan efisien.

Landing page ini dibangun menggunakan **Next.js 16**, **React 19**, **TypeScript**, **Tailwind CSS v4**, dan **Framer Motion** sehingga menghasilkan tampilan yang modern, responsif, ringan, dan cepat.

## 🚀 Installation

Clone repository

```bash
git clone https://github.com/themufid/jadwal-pintar-landingpage.git
```

Masuk ke folder project

```bash
cd jadwal-pintar-landingpage
```

Install dependencies

```bash
pnpm install
```

Jalankan development server

```bash
pnpm dev
```

Buka browser

```text
http://localhost:3000
```

---

## 🏗️ Build Production

```bash
pnpm build
```

Menjalankan hasil build

```bash
pnpm start
```
